# Instalacion
Se necesita tener las herramientas stack y cabal para poder utilizar el proyecto, una vez
instaladas se ejecuta de la siguiente manera:

> stack init

> stack build

> stack run -- < bandera >


Para ver todas las bandera utilizables ejecutar 
> stack run -- -h
